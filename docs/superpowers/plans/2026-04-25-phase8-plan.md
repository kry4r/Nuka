### 8.1.c Wire into App.tsx + HUD/StatusBar/MessageRow
**Files:** `src/tui/App.tsx`, plus the three consumer components. Tests: extend `test/tui/Status/Hud.test.tsx` to assert color flows through.

### 8.2.a Aggregator + chart
**Files:** `src/core/stats/{aggregate,chart}.ts`. Tests: `test/core/stats/{aggregate,chart}.test.ts`.

`chart(byModel, width)` returns the ASCII bar lines.

### 8.2.b `/stats` slash + view
**Files:** `src/slash/stats.ts`, `src/tui/Stats/{StatsView,RangeTabs}.tsx`. Tests: `test/tui/Stats/StatsView.test.tsx`.

## M2 — Rewind + Plan-mode

### 8.3.a Truncate API + slash UI
**Files:** `src/core/session/manager.ts` add `truncateAfter(messageId)`; `src/slash/rewind.ts`; `src/tui/Rewind/MessageSelector.tsx`. Tests: `test/core/session/truncate.test.ts`, `test/tui/Rewind/MessageSelector.test.tsx`.

### 8.3.b File-checkpointing scaffolding (off by default)
**Files:** `src/core/rewind/{checkpoint,restore}.ts`. Tests: `test/core/rewind/checkpoint.test.ts`.

Snapshot SHA1 of files mentioned in `Write`/`Edit` tool calls per turn; gated by `config.rewind.fileCheckpointing`. `restore(turn)` is a stub returning `{ ok: false, reason: 'fileCheckpointing disabled' }` when the flag is off — never executes destructive ops.

### 8.4.a Permission-mode='plan' gate
**Files:** `src/core/permission/checker.ts` (extend). Tests: extend `test/core/permission/checker.test.ts`.

When `mode === 'plan'`, reject `Write`/`Edit`/`Bash` and any MCP/plugin tool with `annotations.destructive || annotations.openWorld`.

### 8.4.b Plan storage + slash + system-prompt injection
**Files:** `src/core/plan/state.ts`, `src/slash/plan.ts`, `src/core/agent/systemPrompt.ts` (`## Plan` injection). Tests: `test/core/plan/state.test.ts`, `test/slash/plan.test.ts`.

`/plan on|off|show|write <text>|apply`.
